<big><strong>定时器除了使用内部时钟线（APB1/APB2）的电平信号进行计数，还能引入外部信号，通过外设输出的电平信号进行计数。</big></strong><br>
<big><strong>本文中，采用TCRT5000（红外反射光电开关），作为外部信号的输入。</big></strong><br>

---
# 芯片配置
> [!note]
> STM32F103C8T6芯片中仅TIM1和TIM2定时器支持外部输入，这会随芯片型号而变化。

1. 在左侧**Timers**中选择**TIM2**，在右侧**Clock Source**中选择**ETR2**（外部时钟模式2）；<br>
2. 可以看见右侧PA0引脚已被启用为**TIM2_ETR**，现在PA0引脚的高低电平将会接入到触发控制器中，并控制计数器，而非原来的内部时钟。<br>

---
# 代码配置
<big><strong>目的：TCTR5000每次检测到变化时，将其存入定时器TIM2的计数器并通过串口发送回上位机。</big></strong><br>
通过宏函数`__HAL_TIM_GET_COUNTER`直接读取**计数器寄存器**中的数值，若产生变化，则通过USRT2串口将其返回上位机：<br>
```c
HAL_TIM_Base_Start(&htim2);
int counter=0,counter_temp=-1;
char message[20]="";
while (1)
  {
    counter=__HAL_TIM_GET_COUNTER(&htim2);
    if(counter!=counter_temp){
      counter_temp=counter;
      sprintf(message, "counter:%d",counter);
      HAL_UART_Transmit_IT(&huart2, message, strlen(message));
    }
  }
```
将手指反复扫过TCTR5000，可以在串口助手中看到`counter`数值的递增：

# 外部时钟的多种线路
<img src="/images/posts/STM32/STM32-7/tim-etr2.png" width="400" height="400">

ETR经过**极性选择**，**边沿检测**，**预分频**，**输入滤波**：<br>
- **极性选择**：可将**上升沿**和**下降沿**互换；<br>
- **边沿检测**：只能进行**上升沿**检测，但可以搭配**极性选择**实现**下降沿**检测；<br>
- **预分频**：降低ETR的频率；<br>
- **输入滤波**：过滤抖动信号，在CubeMX设置的**Trigger Filter**即是在设置此功能。<br>

之后分为两支：<br>
- **ETR2**：直接抵达**触发控制器**，属于**外部时钟模式2**；<br>
- **ETR1**：连接到**触发器**后经过选择再连接到**触发控制器**，属于**外部时钟模式1**。<br>

除了**外部时钟模式2**这条线路，还有**外部时钟模式1**，它会通过**触发器**选择一个信号，然后进入到**触发控制器**里面的**从模式控制器**。<br>
- 在CubeMX中**Slave Mode**中选择**External Clock Mode 1**；<br>
- 在**Trigger Source**中选择不同的触发方式。<br>

以STM32F103为例，每个定时器有四个输入信号，**TI1**到**TI4**（TIM Input），其中**TI1**和**TI2**与触发控制器有关；<br>
每条TI信号线会依次经过**输入滤波器**和**边沿检测器**：<br>
- **输入滤波器**：过滤抖动信号；
- **边沿检测器**：可设置为**上升沿触发**或者**下降沿触发**，此外还能设置为**双边沿触发**，检测到上升沿/下降沿时输出一个脉冲信号。

之后会继续分出**TI1FP1**，**TI1FP2**，**TI2FP1**，**TI2FP2**，以及**TI1_ED**，其中**TI1FP1**，**TI2FP2**，**TI1_ED**和**ETR1**共同接入到一个**触发器**，并选择其中一个信号接入到**触发控制器**中的**从模式控制器**。<br>
<small>其中，**TI1_ED**固定只能为**双边触发**。</small>

> [!note]
> 更改不同的触发模式后，对应的GPIO口也会发生变化。
